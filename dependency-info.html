Dependency Information

Apache Maven

<dependency>
  <groupId>org.mapdb</groupId>
  <artifactId>mapdb</artifactId>
  <version>1.1.0-SNAPSHOT</version>
  <type>bundle</type>
</dependency>

Apache Buildr

'org.mapdb:mapdb:bundle:1.1.0-SNAPSHOT'

Apache Ant

<dependency org="org.mapdb" name="mapdb" rev="1.1.0-SNAPSHOT">
  <artifact name="mapdb" type="bundle" />
</dependency>

Groovy Grape

@Grab(group='org.mapdb', module='mapdb', version='1.1.0-SNAPSHOT')

Leiningen

MapDB provides concurrent Maps, Sets and Queues backed by disk storage or off-heap memory. It is a fast and easy to use embedded Java database engine. And it is completely free under Apache License. MapDB is free as speech and free as beer under Apache License 2.0.
